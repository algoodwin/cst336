Abstract Data Structures Description
Linked List Linked Lists are linear data structures made up of nodes. Nodes hold their own address and data and can "point" to the next node to create a sequence of relative data
Binary Tree Binary Trees consist of linked data with every node being connected to the parent(or root) node of the tree. Binary tree's only contain up to two children, one right child and one left child.
Stack Stacks are container objcets that follow last in first out rule. Meaning the last data point that is popped (added) to a stack is the first data point to be pushed off the data set.
Queue Queues are container objects that follow the first in last out rule. Meaning the first object that is enqueued (added) is the first object dequeued (taken off) of the data set.
